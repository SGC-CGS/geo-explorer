How to use

How to Use This Application

The dropdown lists can be used to selected the indicator to draw on the map. The map will be drawn once an indicator is selected and the 'update map' button is activated.

The map is interactive, the user can click on polygons to obtain more information about the corresponding data point.

The user can navigate the map in the following ways:

  • Clicking and dragging the map will pan the map;
  • Scrolling the mouse wheel will zoom in or out of the map;
  • Clicking the 'Zoom In' button ( ) or the 'Zoom Out' button ( ) buttons will zoom in or out, respectively; or
  • Clicking the 'Home' button ( ) will return the map to its default extent.

The 'Full Screen' button ( ) will open the map in full screen mode.

The 'Legend' button ( ) will toggle a legend widget with information on the color ramps used to draw the map.
